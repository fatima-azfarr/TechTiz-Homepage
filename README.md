TechTiz Homepage

This is the homepage design for TechTiz, created as part of my frontend development learning journey.
The project focuses on building a clean and modern UI using basic web technologies.

Technologies Used
	•	HTML5
	•	CSS3
	•	JavaScript

Features
	•	Responsive homepage layout
	•	Clean UI design
	•	Smooth styling and structured sections
	•	Basic JavaScript functionality for interactivity

Purpose of the Project

This project was created to practice and improve my skills in:
	•	Page structuring with HTML
	•	Styling and layout with CSS
	•	Adding interactivity using JavaScript

How to Run the Project

1.Clone the repository: git clone https://github.com/fatima-azfarr/TechTiz-Homepage.git
2.  Navigate into the project folder: cd TechTiz-Homepage
3.  Open index.html in your browser: open index.html  # Mac or start index.html # Windows
